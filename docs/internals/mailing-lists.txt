=======================
Mailing lists and Forum
=======================

.. Important::

    Please report security issues **only** to
    security@gingersociety.org.  This is a private list only open to
    long-time, highly trusted GingerDJ developers, and its archives are
    not public. For further details, please see :doc:`our security
    policies </internals/security>`.

GingerDJ Forum
============

GingerDJ has an `official Forum`_ where you can input and ask questions.

There are several categories of discussion including:

* `Using GingerDJ`_: to ask any question regarding the installation, usage, or
  debugging of GingerDJ.
* `Internals`_: for discussion of the development of GingerDJ itself.

.. _official Forum: https://forum.gingerdj.gloportal.dev
.. _Internals: https://forum.gingerdj.gloportal.dev/c/internals/5
.. _Using GingerDJ: https://forum.gingerdj.gloportal.dev/c/users/6

In addition, GingerDJ has several official mailing lists on Google Groups that
are open to anyone.

.. _gingerdj-users-mailing-list:

``gingerdj-users``
================

.. note::

    The `Using GingerDJ`_ category of the `official Forum`_ is now the preferred
    venue for asking usage questions.

This is the right place if you are looking to ask any question regarding the
installation, usage, or debugging of GingerDJ.

.. note::

    If it's the first time you send an email to this list, your email must be
    accepted first so don't worry if :ref:`your message does not appear
    <message-does-not-appear-on-gingerdj-users>` instantly.

* `gingerdj-users mailing archive`_
* `gingerdj-users subscription email address`_
* `gingerdj-users posting email`_

.. _gingerdj-users mailing archive: https://groups.google.com/g/gingerdj-users
.. _gingerdj-users subscription email address: mailto:gingerdj-users+subscribe@googlegroups.com
.. _gingerdj-users posting email: mailto:gingerdj-users@googlegroups.com

``gingerdj-developers``
=====================

.. note::

    The `Internals`_ category of the `official Forum`_ is now the preferred
    venue for discussing the development of GingerDJ.

The discussion about the development of GingerDJ itself takes place here.

Before asking a question about how to contribute, read
:doc:`/internals/contributing/index`. Many frequently asked questions are
answered there.

.. note::

    Please make use of
    :ref:`gingerdj-users mailing list <gingerdj-users-mailing-list>` if you want
    to ask for tech support, doing so in this list is inappropriate.

* `gingerdj-developers mailing archive`_
* `gingerdj-developers subscription email address`_
* `gingerdj-developers posting email`_
